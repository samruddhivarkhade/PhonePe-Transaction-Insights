
# PhonePe Transaction Insights Dashboard

An interactive **Streamlit** dashboard to visualize PhonePe transaction data. Explore transaction amounts, counts, and yearly trends by category. The dashboard allows users to filter data by year and category and gain meaningful insights into transaction trends over time.

## Features

- **Dynamic Filtering:** Filter transactions by **Year** and **Category**
- **Data Visualization:**
  - Total Transaction Amount by category
  - Transaction Count by category
  - Yearly Transaction Growth trends
- **Summary Metrics:** Quick overview of total transaction amount and count
- **Interactive Dashboard:** Built with **Streamlit**, **Seaborn**, and **Matplotlib**
- **Data Sources:** Supports CSV or MySQL database connection
- **Responsive Design:** Works on desktop and mobile browsers
- **Easy Deployment:** Deployable on **Streamlit Cloud** or any web server
